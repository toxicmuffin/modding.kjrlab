LOOT Setup Guide

Introduction

Load Order Optimisation Tool (LOOT) is an essential application for managing your mods' load order, resolving conflicts, and providing valuable information about your mods. This guide will assist you in installing, setting up, and using LOOT with Mod Organizer 2 (MO2).

Installation

  1. Download the latest version of LOOT from the official LOOT website.
  2. Once downloaded, run the installer and follow the instructions. Make sure you install LOOT in a dedicated folder outside of your Program Files directory to avoid potential permission issues.
  3. After installation, run LOOT once to generate necessary files and close it.

Setup with Mod Organizer 2

  1. Open MO2 and click the dropdown arrow next to the Run button at the top right.
  2. Select 'Edit...' from the dropdown menu.
  3. In the Modify Executables window, under Title, enter 'LOOT'.
  4. Next to the Binary field, click the folder icon and navigate to your LOOT installation folder. Select the LOOT application and click 'Open'.
  5. Click 'Add' to add LOOT to your list of executables in MO2, then click 'Close'.
  6. LOOT is now set up to run from within MO2. You can launch it using the dropdown menu next to the Run button.

Using LOOT

  1. With your mods installed through MO2, launch LOOT from the MO2 interface.
  2. Before sorting your mods, it's crucial to update the master list. The master list contains metadata for many mods and helps LOOT determine the optimal load order. To do this, select 'Update Masterlist'. LOOT will then fetch the latest version of the master list from the online repository.
  3. After updating the master list, click the 'Sort Plugins' at the top.
  4. LOOT will analyze your mods and provide a suggested load order. Click 'Apply' to accept LOOT's suggested load order.
  5. Close LOOT and return to MO2. Your plugins will now be arranged according to LOOT's recommendations.

It's advised to run LOOT and update the master list whenever you install new mods or periodically, to ensure the load order remains optimal and to reduce the risk of conflicts. The master list is continually updated by the community to reflect the latest compatibility information between mods.

Troubleshooting

If you encounter any problems while using LOOT, refer to the official LOOT FAQ or seek help from the LOOT community on Reddit.

Mod Organizer 2 Setup Guide

Introduction

Mod Organizer 2 (MO2) is a powerful mod management tool for games built on the Creation Engine. This guide will help you install and setup MO2 for optimal use.

Installation

  1. Download the latest version of Mod Organizer 2 from GitHub.
  2. Extract the downloaded .zip file using a tool like 7-Zip or WinRAR.
  3. Navigate to the extracted folder and run ModOrganizer.exe to start the application.

Initial Setup

Setting up Mod Organizer 2 (MO2) properly is important to ensure a smooth modding experience. Here's a detailed guide on how to do it:

  1. Create a Separate Modding Folder

    It's recommended to create a dedicated folder for your modding activities, ideally not on your OS drive (typically drive C:). This preserves your data if you ever need to reinstall your OS and helps keep your modding files isolated. An example would be something like D:\Modding or E:\Games\Mods.

  2. Run ModOrganizer.exe

    Navigate to the folder where you extracted MO2 and run ModOrganizer.exe. This will initiate the first-run setup process.

  3. Select the Game

    MO2 will ask you to select the game you want to manage. If your game is not detected automatically, you'll need to navigate to its installation folder manually. Make sure that you select the right game, as MO2 uses game-specific settings and features.

  4. Choose the Mode

    You'll be prompted to choose between Portable and Instance mode. For most users, Portable mode is recommended. This mode keeps all MO2 settings and files in one location, which is easier to manage and backup.

  5. Set the Base Directory

    In the next step, MO2 will ask for a base directory. This is where MO2 will store its data, including mods, profiles, and overwrite data. You should choose the dedicated modding folder you created in the first step. Note that each game managed by MO2 should have its own base directory to avoid conflicts.

  6. Finish the Setup

    Once you've made your selections, click 'OK' to finish the setup. MO2 is now ready to manage your game's mods. All mods installed through MO2 will be stored in the base directory, keeping your game folder clean.

Troubleshooting

If you run into any issues while using Mod Organizer 2, refer to the official MO2 wiki or seek help from the MO2 community on Reddit.

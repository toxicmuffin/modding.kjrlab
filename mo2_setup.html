Mod Organizer 2 Setup Guide

Introduction

Mod Organizer 2 (MO2) is a powerful mod management tool for games built on the Creation Engine. This guide will help you install and setup MO2 for optimal use.

Installation

  1. Download the latest version of Mod Organizer 2 from GitHub.
  2. Extract the downloaded .zip file using a tool like 7-Zip or WinRAR.
  3. Navigate to the extracted folder and run ModOrganizer.exe to start the application.

Initial Setup

Setting up Mod Organizer 2 (MO2) properly is crucial to ensure a smooth modding experience. Here's a detailed guide, including best practices for organizing your setup:

  1. Create a Dedicated Modding Folder

    It’s strongly recommended to create a dedicated folder for your modding activities, ideally not on your OS drive (usually C:). This keeps your modding files isolated and safe if you need to reinstall your OS. For example, you can use a structure like G:\Modding\Skyrim\modpack1.

  2. Install MO2 as a Portable Instance

    For simplicity and better organization, install MO2 in a dedicated folder within your modding directory, such as G:\Modding\Skyrim\modpack1. Using a portable instance keeps all MO2 files, settings, and mods contained in one location, making backups and maintenance easier.

    Note: MO2 allows only one portable instance per installation. If you plan to manage multiple setups for the same or different games, create a new portable instance in a separate folder (e.g., G:\Modding\Skyrim\modpack2).

  3. Run ModOrganizer.exe

    Navigate to the folder where you installed MO2 and run ModOrganizer.exe. Follow the first-run setup prompts to configure MO2 for your game.

  4. Select the Game

    MO2 will ask you to select the game you want to manage. If your game isn’t detected automatically, navigate to its installation folder manually. Choose the correct game to ensure proper compatibility and functionality.

  5. Set the Base Directory

    During setup, MO2 will ask for a base directory where it will store mods, profiles, and other data. Point this to the folder you created earlier, such as G:\Modding\Skyrim\modpack1. Each game or setup should have its own base directory to avoid conflicts.

  6. Understand Profile Limitations

    While MO2 supports profiles for managing different setups within the same instance, profiles can become cluttered if you have a large mod list, as all mods will appear in the list, even if they’re not active. If you frequently use similar mods across profiles, this may be manageable. However, for entirely separate modding setups, creating a new portable instance is recommended.

  7. Complete the Setup

    Once you’ve configured your settings, click OK to finish the setup. MO2 is now ready to manage your mods. Remember, installing mods through MO2 keeps your game folder clean, which simplifies troubleshooting and maintenance.

Recommended Things to Do After Installation

  1. Enable Dark Mode

    MO2 supports a dark mode interface, which is easier on the eyes, especially during long modding sessions. To enable dark mode:

    1. Open MO2 and go to Settings by clicking the gear icon in the top menu.
    2. Navigate to the Theme tab.
    3. Choose the Dark theme from the dropdown menu.
    4. Click Apply to save your changes.

  2. Use Profile-Specific Save Games and INI Files

    To ensure your save files and INI configurations are kept separate for each profile:

    1. In MO2, click the Profiles dropdown in the top menu.
    2. Click Manage Profiles.
    3. Ensure the options Use profile-specific save games and Use profile-specific ini files are checked.
    4. Click Close to save your settings.

    These settings prevent conflicts between different setups and ensure a clean and isolated environment for each profile.

Troubleshooting

If you run into any issues while using Mod Organizer 2, refer to the official MO2 wiki or seek help from the MO2 community on Reddit.
